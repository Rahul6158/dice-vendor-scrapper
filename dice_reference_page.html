ABOUT THE ROLE

Oracle Database & System Administrator to support the DMAS accounting system, Oracle EBS (GL, AP, AR) and the 25 Oracle APEX applications and the underlying Oracle Databases. This includes quarterly security patching, system and database upgrades, problem resolution and deployments. This role requires knowledge of Oracle EBS and Oracle database features. Experience with applications developed using Oracle APEX versions 21and 24 and workflow features. This position may require some night and weekend work due to patching.
